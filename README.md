# zktls-reactnative-core-sdk

Primus zkTLS react-native algorithm sdk

## Installation

```sh
npm install zktls-reactnative-core-sdk
```

## Usage


```js
import { PrimusCoreTLS } from 'zktls-reactnative-core-sdk';

// ...
const zkTLS = new PrimusCoreTLS();
// ...

```
